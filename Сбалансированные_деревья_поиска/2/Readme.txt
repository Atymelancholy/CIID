3.3.29

Оптимальный объем памяти. Измените класс RedBlackBTS, чтобы не использовать дополнительную память для бита цвета, с помощью следующего приема:
чтобы закрасить узел красным цветом, обменяйте его ссылки. Тогда для проверки, что узел красный, достаточно проверить, что сго левый дочерний узел больше правого. 
При этом потребуется изменить сравнения, чтобы они учитывали возможный обмен ссылок, и этот прием заменяет сравнение бита сравнением
ключей, что может быть сложнее. Но он демонстрирует, что при необходимости бит цвета можно убрать из узлов

E = 60 минут
Фактическое время = 50 минут
